**Pillar 1: Unified Neural-Constrained Encoding (SCONE + DNA Diamond)**

This breakthrough directly addresses the readout challenges that previously limited composite-letter DNA storage.

**Pillar 2: Breakthrough Error Correction (Gungnir + THEA-code + Self-Synchronizing SDC)**

The **Gungnir** codec, published in *Nature Communications* (April 2026), introduces a proof-of-work paradigm for DNA error correction. With a hash signature for each data fragment, Gungnir corrects substitution, insertion, and deletion errors by testing educated guesses until the hash signature is matched. It achieves **complete binary data recovery from a single sequence copy containing 20% erroneous bases**, requiring neither redundant sequence copies nor high-coverage sequencing—dramatically reducing the overall cost of DNA storage.

**THEA-code**, presented at AAAI 2026, provides an autoencoder-based approach that efficiently generates IDS-correcting codes for complex insertion/deletion/substitution channels. Using disturbance-based discretization and a simulated differentiable IDS channel, THEA-code produces channel-customized codes with commendable performance across realistic DNA storage channels. The encoder maps the source DNA sequence into a longer codeword sequence, and the method follows a heuristic end-to-end autoencoder design.

**Self-synchronizing single-deletion-correcting (SDC) codes** address the self-synchronization problem in DNA coding without using delimiters between codewords. These algorithms detect single-deletion errors while maintaining transmission efficiency, solving a fundamental challenge in DNA data storage.

**Pillar 3: Unconditional Post-Quantum Security (Chemical Functions + pH-Regulated Firewall)**

The **Chemical Functions** framework provides a unified security model that treats chemical systems as noisy challenge–response primitives. Instantiated with two existing DNA-based constructions (operable random DNA and Genomic Sequence Encryption), this framework formalizes the associated chemical function infrastructure for molecular cryptography.

The **pH-regulated camouflage strategy** uses DNA triplex strands to construct a two-layer DNA molecular firewall with four differentiated access modes. As solution pH oscillates, the DNA triplex forms and dissociates, enabling switchable cascaded logic operations that prevent unauthorized access and conceal true input addresses. This represents a fundamental advance in DNA molecular cryptography: encryption concealment that adapts to environmental conditions, protecting against both external attacks and information leakage in dynamic scenarios.
