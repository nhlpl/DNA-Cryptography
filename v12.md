**Pillar 1: End-to-End Neural-Constrained Encoding (SCONE + DNA Diamond)**

The **SCONE** architecture collapses latent compression and DNA encoding into a single step, performing quaternary arithmetic coding directly on the latent space in DNA bases. Its Constraint-Aware Adaptive Coding module dynamically steers the entropy encoder's learned probability distribution to enforce GC balance and homopolymer suppression deterministically during encoding, eliminating post-hoc correction with **<2% latency overhead**. The **DNA diamond** model formulates a decomposable composite letter constellation of 15 letters, enabling **3.125 bits per letter** payload density. The eight-letter system achieves 2.5 bits per letter, while the 15-letter composite achieves error-free recovery at 33× coverage with an entropy-guided two-stage detection strategy for reliable readout at low sequencing coverage. This breakthrough directly addresses the readout challenges that previously limited composite-letter DNA storage.

**Pillar 2: Breakthrough Error Correction (Gungnir + THEA-code + Self-Synchronizing SDC)**

The **Gungnir** codec introduces a proof-of-work paradigm for DNA error correction. With a hash signature for each data fragment, Gungnir corrects substitution, insertion, and deletion errors by testing educated guesses until the hash signature is matched. It achieves **complete binary data recovery from a single sequence copy containing 20% erroneous bases**, requiring neither redundant sequence copies nor high-coverage sequencing—dramatically reducing the overall cost of DNA storage. Gungnir also considers biochemical constraints including GC-content, homopolymers, and error-prone motifs during encoding.

**THEA-code** provides an autoencoder-based approach that efficiently generates IDS-correcting codes for complex insertion/deletion/substitution channels. Using disturbance-based discretization and a simulated differentiable IDS channel, THEA-code produces channel-customized IDS-correcting codes with commendable performance across realistic DNA storage channels.

**Self-synchronizing single-deletion-correcting (SDC) codes** address the self-synchronization problem in DNA coding without using delimiters between codewords. Inspired by Francis Crick's "codes without commas" proposal for DNA sequences, these algorithms detect single-deletion errors while maintaining transmission efficiency, distinguishing the beginning and end of each codeword even in the presence of deletion errors.

A comprehensive systematic benchmarking study has compared state-of-the-art codec implementations, demonstrating that under realistic conditions, **storage densities as high as 117 EB g⁻¹** are feasible using existing codecs and current synthesis/sequencing technologies.

The **Chemical Functions** framework provides a unified security model that treats chemical systems as noisy challenge–response primitives.
